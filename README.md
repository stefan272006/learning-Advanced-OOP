# learning-Advanced-OOP

## About This Repository

This repository is dedicated to learning and practicing advanced Object-Oriented Programming (OOP) concepts for **EECS 2030** at York University.

## Course Information

- **Course:** EECS 2030 - Advanced Object Oriented Programming
- **Institution:** York University
- **Purpose:** Learning and practicing OOP concepts including:
  - Inheritance
  - Polymorphism
  - Encapsulation
  - Abstraction
  - Design patterns
  - Testing and debugging

## Contents

This repository contains code examples, exercises, and projects related to course material.

## Technologies

- Java (OpenJDK 25.0.2)
- Git/GitHub for version control
